What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage.

With CSS you can control the color font, the size of text, the spacing between elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Tip: The word cascading means the a style applied to a parent element will also to all children elements within the body will also get the same color (unless you specify something else)!
